# 144Hz Auto Switcher

Аддон для World of Warcraft 5.4.8, который автоматически переключает частоту обновления монитора между 120Hz и 144Hz в зависимости от режима отображения.

## Возможности

- **Автоматическое переключение**: При переходе в полноэкранный режим частота увеличивается до 144Hz, в оконном режиме возвращается к 120Hz
- **Мгновенное применение**: Использует `RestartGx()` для применения изменений без перезапуска игры
- **Фоновый мониторинг**: Проверяет режим отображения каждые 5 секунд
- **Ручное управление**: Команда `/144hz` для проверки текущего состояния

## Установка

1. Скопируйте папку `144hz` в `World of Warcraft\Interface\AddOns\`
2. Перезапустите игру
3. Аддон автоматически начнет работу

## Использование

Аддон работает автоматически после загрузки. При изменении режима отображения (полноэкранный/оконный) частота обновления изменится автоматически.

### Команды

- `/144hz` или `/hz` - показать текущий режим отображения и частоту обновления

## Технические детали

- **Полноэкранный режим**: 144Hz (`SET gxRefresh "144/1"`)
- **Оконный режим**: 120Hz (`SET gxRefresh "119/1"`)
- Изменения сохраняются в `Config.wtf` и применяются мгновенно

## Требования

- World of Warcraft 5.4.8 (Mists of Pandaria)
- Монитор с поддержкой 144Hz
